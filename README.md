# DingTalkFucker

## 安装

需要安装 [MonkeyDev](https://github.com/AloneMonkey/MonkeyDev)

## 功能

* 支持位置 wifi 和 gps 定位打卡， 
* 支持延迟红包，对红包消息进行正则过滤，可限定红包额度大小
